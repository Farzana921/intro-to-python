# Week 01 – Python Basics

This folder contains my **first week of Python practice**.  
The programs focus on learning basic Python concepts such as input, variables, calculations, and output.

---

## Files

- **first_project.py**  
  A simple program that asks for the user’s name and prints a welcome message.

- **Split_the_Bill.py**  
  A small calculator that splits a bill between people, including a tip.

- **Tech_Company.py**  
  A fun program that generates a tech company name using user input.

---

## Concepts Practiced

- User input (`input`)
- Variables
- Basic math
- String formatting
- Printing output

---

## Goal

To build a strong foundation in Python by practicing simple programs each week.
